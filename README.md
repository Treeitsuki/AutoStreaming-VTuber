# Running code
1. Clone my repository
    ```
    git clone https://github.com/Treeitsuki/AutoStreaming-VTuber.git
    ```
1. Set up [VMagicMirror](https://malaybaku.github.io/VMagicMirror/)

    https://malaybaku.github.io/VMagicMirror/download/

1. Run ShellScript
    ```
    bash main.sh
    ```
